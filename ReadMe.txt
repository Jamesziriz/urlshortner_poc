Hosted at https://ziriz.herokuapp.com/

Prerequisites
1.Node to be installed to run locally
2.MONGODB_URI from MongodbAtlas

Steps to run locally
1.Clone the repo
2.Create .env file and store the MONGODB_URI
3.Run 'npm install'
4.Run 'npm start'

Steps to deploy on cloud 
1.Upload files to respective cloud 
2.Add MONGODB_URI to config vars as key-value pair
